# 전국 교량 검색 시스템
WPF Project (Korea Bridge Searching System_ 전국 교량 검색 시스템)  
전국의 교량을 지역, 교량의 종류, 길이를 특정해 검색할 수 있는 시스템  

## 사용 기술
DB: MongoDB  
Layout, Logic: WPF, C#  

## 업무 분담  
Server : 김현석  
Layout : 김재연
